# MERN_RECIPE_CREATE
A full-stack recipe manager built using MongoDB, Express, React, and Node.js.

## Features
- Add, update, delete recipes
- User-friendly interface
- MongoDB backend

## Tech Stack
MERN (MongoDB, Express, React, Node)

## Run Locally
1. Clone this repo
2. `cd client && npm start`
3. `cd server && npm start`
